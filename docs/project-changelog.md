## 2026-05-13 — Initial Release

### Added
- Daily outfit suggestion page with deterministic date-based selection
- Shuffle functionality with 2-piece, 3-piece, and all-outfit filters
- "Wear Today" toggle with pop/unpop animations
- Worn history tracking (last 5 outfits excluded from daily suggestion)
- Collection browser page with outfits grouped by shirt color
- Piece-count filter chips on collection page
- Automatic light/dark theme switching (6am-6pm boundary)
- Manual theme toggle with localStorage persistence
- 57 curated outfits with composite images
- Responsive layout with 3 breakpoints (640px, 720px, 960px)
- Cormorant Garamond serif typography for old-money aesthetic
- GitHub Pages deployment
